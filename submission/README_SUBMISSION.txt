SJC Threads — what to put in your ZIP
======================================

1. Source code
   - Zip the whole project folder (recommended: exclude node_modules/ unless your instructor wants it).

2. Data file (.json)
   - Include: submission/sjc_threads_db_v1.json
   - Note: When you run the app on a device/simulator, it reads/writes its own copy under app documents as sjc_threads_db_v1.json. The submission folder sample matches that schema for grading.

3. Design documentation (1–2 pages)
   - Include DESIGN_DOCUMENTATION.md from the project root.
   - If PDF is required, export or print that Markdown to PDF.
